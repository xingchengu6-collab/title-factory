# 标题工厂运营计划

目标：把标题工厂从工具站做成可持续增长的被动型数字资产。

## 当前定位

标题工厂不是泛 AI 写作工具，而是面向内容创作者、小商家、运营人员的“标题与文案增长工具”。

核心资产：

- 免费工具：吸引搜索和收藏
- AI 工具工作台：承接最有可能付费的早期用户
- 125 个 SEO 长尾页：承接被动流量
- 25 个行业模板包页：承接更明确的购买意图
- 12 个高购买意图方案页：承接“模板包/销售页/提示词”等更接近付费的搜索
- 免费模板包：建立信任
- 模板包 / Pro：承接付费
- 运营看板：判断哪些方向值得加码

## 当前实际状态

- 网站已部署到 GitHub Pages。
- 付费模板包交付文件已生成，但保存在 `private-delivery/`，不进入公开仓库。
- 面包多创作者申请已提交，当前等待平台审核；审核通过前不能创建商品。
- 免费样品包已有独立页面和 CSV 下载，用来承接暂时无法付款的访客。
- 行业模板包入口已拆成 25 个独立页面，导向免费工具、样品包和完整付费包。
- 解决方案入口已拆成 12 个独立页面，导向免费工具、行业包、样品包和完整付费包。

## 商业路径

1. 免费工具获取流量。
2. 模板包验证低价付费。
3. Pro 版验证订阅意愿。
4. 把表现最好的行业做深，变成行业内容工作台。

## 30 天执行节奏

第 1 周：上线和收录

- 绑定正式域名
- 用正式域名生成 sitemap
- 提交搜索引擎
- 接入真实 AI 密钥
- 等面包多审核通过后配置模板包支付链接

第 2 周：观察数据

- 每天看 `/admin.html`
- 记录访问最多的页面
- 记录生成最多的行业
- 记录模板包点击和等待名单提交

第 3 周：做深高潜行业

- 选 3 个表现最好的行业
- 每个行业新增 20 个更具体页面
- 为每个行业补 1 个专属模板包
- 调整首页精选入口

第 4 周：开始收费验证

- 上架 ¥99 模板包
- Pro 等待名单转成早鸟价
- 给 Pro 加第一批功能：批量生成、导出、品牌语气保存
- 用数据决定是否继续扩 SEO 或转向产品功能

## 每周指标

- 页面访问数
- 生成次数
- 下载点击
- 等待名单提交
- 模板包购买
- Pro 转化
- 搜索收录页面数

## 产品升级路线

阶段 1：标题生成器

- 当前已完成
- 目标是验证流量和需求

阶段 2：内容日历工具

- 输入行业和目标用户
- 生成 30 天内容计划
- 支持导出
- 当前已完成基础版

阶段 3：行业内容工作台

- 保存品牌语气
- 批量生成
- 历史记录
- 行业模板包
- AI 工具行业基础版已完成

阶段 4：团队版 / 白标版

- 多成员
- 品牌库
- API
- 可嵌入工具

## 我会优先做的判断

不追求“功能很多”，优先看数据：

- 哪些页面自然有人来
- 哪些行业有人生成
- 哪些入口有人点击下载
- 哪些人愿意留下邮箱

只有被数据证明的方向，才继续加资源。
