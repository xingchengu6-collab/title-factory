# 标题工厂 v1.0 发布说明

## 版本定位

标题工厂 v1.0 是一个可上线的标题与内容生成工具站，包含免费工具、SEO 长尾页、AI 工具垂直工作台、模板包样品、可售卖模板包草案、等待名单、事件记录和运营看板。

## 核心能力

- 6 个生成模式：小红书、短视频、商品标题、朋友圈、提示词、30 天内容日历
- 125 个 SEO 长尾工具页
- AI 工具内容工作台
- 批量生成
- 品牌语气本地保存
- Markdown / CSV 导出
- 免费模板包
- 免费样品包独立页面和 CSV 下载
- AI 工具付费模板包草案
- 付费页信任模块、适合/不适合、FAQ 和结构化信息
- 样品包试用路径和升级引导
- Pro 等待名单
- 运营看板
- 健康检查
- 上线前自动检查

## 上线前必须配置

```text
SITE_URL=https://你的正式域名
AI_API_KEY=你的 AI 密钥
TEMPLATE_PACK_URL=模板包购买链接
PRO_CHECKOUT_URL=Pro 购买或订阅链接
```

## 发布命令

```bash
SITE_URL=https://你的正式域名 npm run generate
npm run preflight
npm start
```

## 发布后第一周观察

- `/admin.html` 的页面访问
- AI 工具工作台访问
- 生成次数
- 模板包点击
- Pro 点击
- 等待名单提交
- 搜索引擎收录数量

## 当前限制

- 面包多创作者申请已提交，审核通过前还不能创建真实商品和支付链接
- 还没有用户账户系统
- 事件和等待名单使用本地 CSV，适合早期验证
- 没有正式域名时，SEO 还不能真正发挥效果
