# 标题工厂入门模板包

这是一份给工具站早期用户的免费样品包。目标不是堆很多模板，而是让用户下载后立刻感到“这个东西有用”，为后续付费模板包或 Pro 版铺路。

## 1. 小红书标题公式

1. 【人群】别再这样做【事情】了
2. 【行业】新手最容易忽略的 5 个细节
3. 我用【工具/方法】解决了【具体痛点】
4. 【行业】内容没人看？先改这 3 个标题
5. 适合【人群】直接套用的【行业】选题
6. 【产品/服务】怎么写才不像广告？
7. 为什么你发了很多内容，还是没人咨询？
8. 【行业】账号低粉也能成交的内容结构

## 2. 短视频开头公式

1. 如果你还在为【痛点】发愁，先看完这条
2. 很多【人群】都做错了第一步
3. 别急着买【产品】，先搞懂这 3 件事
4. 30 秒讲清楚：【行业】怎么做内容才有人问
5. 你以为是流量问题，其实是表达问题

## 3. 商品标题结构

```text
【核心产品】+【目标人群】+【使用场景】+【关键卖点】+【结果/体验】
```

示例：

```text
通勤女生轻便托特包，大容量、防泼水、电脑隔层，适合上班和短途出行
```

## 4. AI 提示词

```text
你是一个中文增长文案编辑。
我的行业是：【填写行业】
目标用户是：【填写用户】
产品/服务/主题是：【填写主题】
用户痛点是：【填写痛点】
请生成 20 个具体、自然、有点击欲望但不夸张的小红书标题。
```
